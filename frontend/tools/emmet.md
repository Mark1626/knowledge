# Emmet Abbreviations

[Back](../../index.md){: .button}

Emmet abbreviations can be used to speed up html, css creation

``` html
<!-- ul.list>li*3.row will generate  -->

<ul class="list">
  <li class="row"></li>
  <li class="row"></li>
  <li class="row"></li>
</ul>
```
